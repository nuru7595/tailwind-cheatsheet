Tailwind Setup Commands

Install "Node.js" before starting

npm init -y

Use in Terminal to create package.json file.

  "start": "vite",
  "dev": "vite build"

Go to "package.json" and replace line no 6.
"test": "echo \"Error: no test specified\" && exit 1"

npm install -D tailwindcss postcss autoprefixer vite

Use in Terminal to install "TailwindCss", "PostCss", "AutoPrefixer" and "Vite".

npx tailwindcss init -p

Use in Terminal to create postcss.config.js and tailwind.config.jsfiles.

  content: ['*'],

Go to "tailwind.config.js" and replace line no 3.
content: [],

@tailwind base;
@tailwind components;
@tailwind utilities;

Create your css file and paste these lines at the top.

npm run start

Use in Terminal to run your page in live server.

npm run start

After completing your project. Use it in Terminal to compile your project.
