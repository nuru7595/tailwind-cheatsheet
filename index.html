Install "Node.js" before starting

Tailwind Setup

npm init -y

Use in Terminal. For creating package.json file.

  "start": "vite"

Go to "package.json" and replace line no 6.
"test": "echo \"Error: no test specified\" && exit 1"

npm install -D tailwindcss postcss autoprefixer vite

Use in Terminal. For installing "TailwindCss", "PostCss", "AutoPrefixer" and "Vite".

npx tailwindcss init -p

Use in Terminal. For creating postcss.config.js and tailwind.config.jsfiles.

  content: ['*'],

Go to "tailwind.config.js" and replace line no 3.
content: [],

@tailwind base;
@tailwind components;
@tailwind utilities;

Now create your css file and paste it.

npm run start

Use in Terminal. To run your page in live server.
