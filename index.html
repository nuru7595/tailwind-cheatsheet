Tailwind Cheatsheet

Step 1: Run in Terminal one by one.

npm init -y
npm install -D tailwindcss postcss autoprefixer vite
npx tailwindcss init -p

Step 2: Replace content:[] in tailwinnd.config.js.

  content: ["*"],

Step 3: Replace full line "test": "echo... in package.json.

    "liveserver": "vite",
    "dist": "vite build",
    "build": "tailwindcss -i style.css -o style.min.css --minify"

Step 4: Add the following codes in Css file.

@tailwind base;
@tailwind components;
@tailwind utilities;

Step 5: Run in Terminal

npm run liveserver

Step 6: Run in Terminal (Build for upload)

npm run build
