Tailwind Setup

Install Node.js before starting the process.

Step 1: Create an Empty Folder and open it in VS Code.

Step 2: Open Terminal and run the following code to create package.json file.

npm init -y

Step 3: Open package.json file and replace line no 6 "test": "echo \"Error: no test specified\" && exit 1" with the following code.

    "watch": "tailwindcss -i ./input.css -o ./output.css --watch"

Step 4: Go to Terminal and run the following code to Install Tailwind Css in your Project.

npm install -D tailwindcss

Step 5: Run the following code again in Terminal to create tailwind.config.js file.

npx tailwindcss init

Step 6: Open tailwinnd.config.js file and select everything and replace with the following codes.

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: ["./*.{html,js}"],
  theme: {
    extend: {},
  },
plugins: [],
}

Step 7: Create a new file, name it input.css, open the file and paste the following codes.

@tailwind base;
@tailwind components;
@tailwind utilities;

Step 8: Create a new file, name it index.html, open the file and paste the following codes.

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <!--~~~~~~~~~~~~~~~ Favicon ~~~~~~~~~~~~~~~-->
        <link rel="shortcut icon" href="assets/img/favicon.ico" type="image/x-icon" />
        <!--~~~~~~~~~~~~~~~ Icon ~~~~~~~~~~~~~~~-->

        <!--~~~~~~~~~~~~~~~ CSS ~~~~~~~~~~~~~~~-->
        <link rel="stylesheet" href="./output.css" />
        <!--~~~~~~~~~~~~~~~ Title ~~~~~~~~~~~~~~~-->
        <title>Website</title>
    </head>
    <body>
        <!--~~~~~~~~~~~~~~~ Header ~~~~~~~~~~~~~~~-->
        <header>
            <h1>Hello World</h1>
        </header>
        <!--~~~~~~~~~~~~~~~ Main ~~~~~~~~~~~~~~~-->
        <main></main>
        <!--~~~~~~~~~~~~~~~ Footer ~~~~~~~~~~~~~~~-->
        <footer></footer>
        <!--~~~~~~~~~~~~~~~ Js ~~~~~~~~~~~~~~~-->
        <script src=""></script>
    </body>
</html>

Step 9: Go to Terminal and run the following code to compile css file named output.css.

npm run watch

Step 10: Now open your project with live server and keep editing.

Never edit output.css file it will automatically genarate your styles.

If you want to upload your project do not upload everything. Upload only index.html, output.css,Js files and Assets.
